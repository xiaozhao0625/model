# P13.5 OCR / Model Runtime Guide

P13.5 starts with planning, manifests, directory rules, health checks, and Web Console visibility. It does not install OCR engines, download model weights, or enable online inference by default.

## Distributed Roles

- M0: central Model Gateway, central OCR, heavy VLM, offline batch analysis.
- W1: PC game capture first. OCR and light models are optional and idle-only.
- W2: local OCR, UI parser, web and PC app analysis, light model priority.
- W3: local OCR, Android UI parser, light model priority. Heavy model work falls back to M0.

## Safety Defaults

- `enabled=false` for every OCR engine and model provider.
- `online_inference_enabled=false`.
- No downloads without source, version, revision, and hash policy.
- Worker inference must be `idle_only` and must not interfere with capture.
- Workers must not connect directly to PostgreSQL.

## Directory Roots

- M0 root: `E:\work`
- W1/W2/W3 root: `D:\work`
- Models: `<root>\models`
- OCR: `<root>\ocr`
- Runtime: `<root>\model_runtime`
