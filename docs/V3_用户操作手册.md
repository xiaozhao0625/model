# V3 用户操作手册

## 用户负责

- 手动安装和启动目标软件或游戏。
- 手动登录账号、处理验证码、人机验证和安全验证。
- 手动配置 OBS，只采目标窗口或游戏画面。
- 在 V3 UI 中配置目标名称、类型、语言、截图来源、保存目录、频率和是否 observe_only。
- 采集时人工观察。

## 用户不需要

- 不需要写代码。
- 不需要手动筛图。
- 不需要配置四机调度。
- 不需要让模型控制电脑。

## 推荐第一轮流程

1. 启动 Master API。
2. 启动 Web Console。
3. 打开 V3 控制台。
4. 创建任务，保持 `observe_only=true`。
5. 使用 folder_watch 或 OBS 产出少量截图。
6. 查看 OCR、模型候选、安全门结果。
7. 人工确认候选框和拒绝原因。
8. 只有在所有 Ready 条件满足后，才考虑启用真实点击。
