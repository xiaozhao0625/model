# V3 模型接入部署与安全边界

## Provider 定位

- PaddleOCR：文字检测、识别、bbox、置信度。
- fastText：可选语言识别 provider。
- ShowUI：主线 UI 模型 provider，用于 UI 区域理解、候选生成、场景分类。
- OmniParser：可选 UI parser，必须通过 license gate。

## 默认状态

所有真实模型默认不自动下载、不自动启用。

`models/model_manifest.example.json` 只提供 schema 和来源示例。真实本地 manifest、权重、缓存不得提交。

## 下载边界

`scripts/v3/model/download_*.ps1` 当前只打印来源、目录和人工确认要求，不会自动联网下载。下载前必须确认：

- 官方来源。
- 版本或 revision。
- 文件清单和预计大小。
- license。
- sha256 或等价完整性记录。

## 动作安全

模型输出只能是：

- 场景分类。
- 候选框。
- 置信度。
- 理由。
- 风险 flags。

模型不能直接输出或执行 click、输入密码、支付、登录、匹配、聊天等动作。所有动作必须经过 Safety Gate。
